1. 打开 generate_icons.html 文件（使用浏览器）
2. 点击"一键生成并下载所有图标"按钮

[English Instructions]
1. Open the generate_icons.html file (using a browser)
2. Click the "Generate and download all icons" button
3. Place the four downloaded icon files in this folder
4. Icon sizes: 16×16, 32×32, 48×48 and 128×128 pixels

Note: All icons are already designed, you just need to click the button to download them.
